## v1.8

> 侧边栏像素图标；连线操作升级；挂机须先确认路线；**浅色主题**；游戏更名 **古海大陆**。

### 本版更新概览

| 类别 | 更新内容 |

| UI 图标 | 侧边栏每项导航增加 **16px 像素图标**（`assets/img/icons/nav/*.svg`） |
| 拖线 | **左键长按拖动** 经过相邻格连线；拖回已选格可缩短路线 |
| 取消 | **右键单击** 某格，取消该格及之后的路径 |
| 挂机门槛 | 须先 **确认路线**（黑→蓝连通），才能点「开始挂机」 |

### 操作流程

1. 地图页 **左键拖线** 从黑格连到蓝格
2. 点 **确认路线** 锁定
3. 点 **开始挂机** — 只打路线上 `?` 标记的怪格
4. 全部清剿 → 地图奖励 → 新图；点 **清除连线** 可重画

## v1.7

> 地图页改为「连线走格」玩法：黑格起点 → 蓝格终点，途中随机遇敌。

### 本版更新概览

| 类别 | 更新内容 |
|------|----------|
| 连线地图 | 点击相邻格连线，从右上黑格连到左下蓝格 |
| 遇敌 | 普通格随机藏怪，进格前不可见，走到才触发战斗 |
| 地区 | 四地区不同网格尺寸与遇敌率（4×5 ~ 5×9） |
| 奖励 | 抵达终点获得地图通关奖励，自动刷新新图 |

### 操作说明

1. 地图页顶部切换地区（需等级解锁）
2. 从 **黑格（右上）** 起，逐格点击 **相邻** 格子连线至 **蓝格（左下）**
3. 点「清除连线」重来；路线合法后点「开始路程」
4. 角色沿路线逐格前进，遇敌格即时战斗，抵达终点领奖

### 钻石获取

| 来源 | 奖励 |
|------|------|
| 挂机战斗 | ~4% 概率 +1 钻石 |
| 连续签到 4~6 天 | 每次 +1 钻石 |
| 连续签到 7 天+ | 每次 +2 钻石 |
| 史诗装备寄售 | ~35% 额外 +1 钻石 |

### 拍卖会

- **竞拍**：每 20 分钟刷新，4 件金币打折装备 + 1~2 件钻石稀世货
- **寄售**：背包装备上拍，金币 = 原价 × 125%（比直接贩卖多 25%）
- 钻石区含：青锋剑拍品、锐眼符、独孤九剑残页等

### 战力公式

综合战力由 HP、六维、暴击、等级加权求和（见 `POWER_WEIGHTS`）。  
换装 / 升级 / 天赋后实时刷新。

**v1.6 = 有钻石花、有拍卖会淘货、战力一目了然。**

### 相对 v1.1 的变更详情

#### 1. 修复主角 HP 不动态掉血

**问题**：怪物 HP 会随战斗减少，主角顶栏 HP 始终满条。

**修复**：
- 新增 `currentHp` 字段，战斗受伤实时扣减
- 顶栏显示 `当前HP / 最大HP`，血条宽度同步
- 战斗页新增 **玩家 vs 敌人** 双栏 HP 对比
- 击败怪物回复 **20% 最大生命**
- 升级时 **生命回满**
- HP 归零：暂停挂机 3 秒，恢复 30% 生命后可再战

#### 2. 宝可梦式六维属性

| 属性 | 说明 | 主要来源 |
|------|------|----------|
| HP | 生命值 | 基础 + 装备 + 武功 + 升级 |
| 物攻 | 物理攻击 | 武器、部分武功 |
| 物防 | 物理防御 | 护甲、饰品 |
| 特攻 | 特殊攻击 | 内功秘籍、法剑 |
| 特防 | 特殊防御 | 护符、轻功 |
| 速度 | 出手顺序 | 轻武器、轻功、饰品 |

**属性公式**：

```
某维最终值 = 基础值 + 装备加成 + 武功加成 + (等级 - 1) × 该维成长
```

**每级成长**：HP+8，物攻/物防/特攻/特防+2，速度+1

**战斗伤害**：
- 物攻：`max(1, 物攻 - 敌物防 + 随机)`
- 特攻：约 25% 概率触发，`max(1, 特攻 - 敌特防 + 随机)`
- 速度高者本回合先行动

#### 3. 打怪掉落系统

| 掉落类型 | 概率 | 说明 |
|----------|------|------|

#### 4. 装备穿戴移至角色栏

- **角色页 → 装备栏位**（武器 / 护甲 / 饰品）
- 点 **更换** → 从背包选择 → **装备**；点 **卸下** 退回背包
- 换装后上方六维属性 **立即更新**
- 背包仅保留材料展示、未穿装备、秘籍学习
