# 剧情大纲 · 古海大陆

> 风格参考：**废柴逆袭 · 异火秘宝 · 宗门恩怨 · 王道热血**（类似斗破式成长，但人物地名均为原创）  
> 游戏内短文本见 `js/data/story.js`

---

## 世界观

大陆盛行 **「气修」**——以丹田之气淬体炼魂，分九段。  
三年前，少年 **陆燃** 还是乌石一带最耀眼的天才，一段气龄便达 **七段**。  
一夜之间，丹田之气尽散，沦为人口中的 **废体**。

村人说他走了邪路，宗门退婚的退婚、踩低的踩低。  
只有青梅竹马 **苏清** 还信他，老村长 **莫伯** 悄悄塞给他一百铜钱：  
「去，把丢掉的尊严，一拳一拳打回来。」

## 主线脉络

```
废体少年 → 村外练级 → 森林异变 → 遗迹寻火 → 苍岚峰证道
   │            │           │            │            │
 序章          第一章       第二章       第三章       第四章
 陆燃          乌石村       幽暗森林     古代遗迹     苍岚峰
```

**核心钩子（斗破式）：**  
陆燃体内并非真的「废了」，而是三年前有一缕 **未觉醒的异种灵火** 在沉睡，  
它吸走了杂气，也在等待真正的机缘——**古遗迹深处的「青岚焰」**。

## 分章剧情

### 序章 · 莫欺少年穷

- 陆燃被赵凌当众嘲讽「三段都不到的废物」。
- 苏清递来伤药，莫伯指路：**村外史莱姆最适合重修打基础**。
- 旁白点题：「今日之辱，他日必百倍奉还。」

### 第一章 · 乌石村（Lv.1+）

- **日常：** 村民冷眼，陆燃默默挂机刷怪、攒铜钱买最低阶剑诀。
- **事件：** 哥布林劫掠粮队，陆燃出手，赵凌却抢功。
- **转折：** 莫伯说森林深处最近 **兽眼发红**，似有人投毒。
- **章末目标：** 升到 Lv.3，踏入幽暗森林。

### 第二章 · 幽暗森林（Lv.3+）

- 野狼、毒蜘蛛异常狂暴，尸体有 **黑色气纹**（血煞帮炼制兽核）。
- 陆燃救下失踪货郎，得线索：有人在遗迹方向筑 **血祭台**。
- **赵凌** 带人进林「猎妖立功」，实则想独占功劳、再踩陆燃一脚。
- **小冲突：** 陆燃与赵凌首次正面交手，险胜或平手——赵凌震惊。
- **章末：** 血煞使逃入遗迹，陆燃追入。

### 第三章 · 古代遗迹（Lv.6+）

- 断壁间 **亡灵不散**，是百年前「青岚剑宗」灭门旧战场。
- 陆燃遇 **墨翁残魂**（小骷髅宠物可在此捕获，暗示为墨翁一缕分神）。
- 墨翁：「你丹田里那东西，是老夫当年见过的 **青岚焰** 火种。」
- 血煞帮欲夺火种献祭，陆燃破阵，觉醒 **异火初兆**（Gameplay：天赋/暴击叙事挂钩）。
- **章末：** 墨翁指向苍岚峰——「赵家背后有人，真正的试炼在峰顶。」

### 第四章 · 苍岚峰（Lv.10+）

- 苍岚峰为剑宗外门试炼之地，登顶者可得 **断峰剑意** 传承。
- **赵凌** 已拜韩宗主门下，放话：「峰顶，你我只活一个。」
- 陆燃一路挂机破敌，体内青岚焰与剑意共鸣，实力暴涨。
- **高潮（预留）：** 峰顶对决赵凌；韩宗主欲收陆燃，陆燃拒：「我的路，自己走。」
- **开放式结局：** 远处传来宗门钟声——**内门选拔** 即将开始（DLC / 后续版本）。
